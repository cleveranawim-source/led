# 🚀 Vercel 배포 가이드

**구글폼/시트 없이** 전부 Vercel 하나에서 동작합니다.
소요시간: **약 15분**

---

## 📋 구조

```
참가자 폰                   Vercel
┌─────────────┐            ┌────────────────────────┐
│  upload.html│ POST ────> │  /api/messages   ──> KV │ (Redis)
│             │ POST ────> │  /api/upload     ──> Blob│ (이미지)
└─────────────┘            └────────────────────────┘
                                    ↑
                                    │ GET (2초마다)
                           ┌─────────────┐
                           │  index.html │ (전광판)
                           └─────────────┘
```

## 1️⃣ 코드 GitHub에 올리기 (3분)

1. 이 프로젝트 파일 전부 다운로드
2. GitHub에 새 repo 생성 (예: `message-board`)
3. 파일 업로드 / push

## 2️⃣ Vercel 프로젝트 만들기 (2분)

1. https://vercel.com/new → GitHub repo 선택 → **Import**
2. Framework: **Other** (자동 감지 됨) → **Deploy**
3. 1분 후 배포 완료. URL 확인 (예: `https://message-board-abc.vercel.app`)

> 이 시점에서 접속하면 "샘플 데이터입니다" 배지가 떠요. API가 아직 KV에 연결 안 됨.

## 3️⃣ Vercel KV 연결 (2분)

1. Vercel 대시보드 → 프로젝트 → **Storage** 탭 → **Create Database**
2. **KV (Upstash for Redis)** 선택 → 이름 아무거나 → **Create**
3. 생성된 KV → **Connect Project** → 이 프로젝트 선택 → **Connect**
4. 자동으로 환경변수(`KV_URL`, `KV_REST_API_*`)가 프로젝트에 주입됨

## 4️⃣ Vercel Blob 연결 (2분) — 이미지용

1. Storage 탭 → **Create Database** → **Blob**
2. 이름 입력 → **Create**
3. **Connect Project** → 이 프로젝트 선택 → **Connect**
4. `BLOB_READ_WRITE_TOKEN` 환경변수 자동 주입

## 5️⃣ 관리자 키 설정 (1분)

삭제 권한용 비밀 키를 수동으로 추가합니다.

1. 프로젝트 → **Settings** → **Environment Variables**
2. 새 변수 추가:
   - Name: `ADMIN_KEY`
   - Value: 아무 문자열 (예: `mysecret2026`) — **이건 외부에 공개 금지!**
   - Environment: Production, Preview, Development 전부 체크
3. **Save**

## 6️⃣ 재배포 (1분)

환경변수는 새 배포에만 반영됩니다.

- Deployments 탭 → 최신 배포 우측 `⋯` → **Redeploy** → 확인

## 🛡️ 관리자 모드 (메시지 삭제)

현장에서 문제가 있는 메시지를 즉시 내리고 싶을 때:

1. 주소창에 `?admin=YOUR_ADMIN_KEY` 붙이기
   예: `https://your-domain.vercel.app/?admin=mysecret2026`
2. URL에서 `?admin=...` 부분은 자동으로 사라지고, **이 브라우저 탭에서만** 관리자 모드 활성화
3. 각 카드 우상단에 빨간 **×** 버튼이 보임 → 클릭 → 확인 → 삭제
4. 상단 "🛡️ 관리자 모드" 배지로 상태 확인
5. 관리자 모드 끄려면 `?admin=` (값 없이) 혹은 탭 닫기

> `ADMIN_KEY`를 모르는 사람은 삭제 불가. URL 파라미터라 **인쇄물/QR로 절대 노출 금지**.

## 💰 비용

전부 **무료 티어**로 운영 가능:
- Vercel Hobby: 무제한 배포
- Vercel KV: 30k commands/월 (2초 폴링 × 참가자 1명 기준 하루 1,728회)
- Vercel Blob: 1GB 저장, 100GB 전송/월

> **참고**: 전광판 화면 하나가 2초마다 폴링 = **하루 43,200회**. 화면을 여러 시간 켜놓으면 KV 무료한도 초과 가능성. 실제 이벤트 몇 시간만 돌리면 문제없음. 장기 운영시 폴링 간격을 5-10초로 늘리기 (`index.html`의 `POLL_MS` 변수).
